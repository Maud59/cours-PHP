- Variables
$name = 'toto';
Le $ permet d'appeler une variable. Une variable n'a pas besoin d'être typée. Une valeur numérique, une chaine de caractères etc. 

La concaténation se fait par un ".". 

Le & permet de cibler une adresse mémoire. 

$name ='toto';
$toto='Chuck Norris';
$$name
Le premier 'toto' devient variable. Le $$name affiche donc la valeur de $name=$toto='Chuck Norris'.

$$name sera égal à ${$name}.

Chaine de caractère : 
$name = 'toto'
echo 'Bonjour $name'
=> affichera Bonjour $name

$name = 'toto'
echo "Bonjour $name"
=> affichera Bonjour toto

Les doubles "" permettent d'interprêter la valeur de la variable. 

- Tableau 

$arr = array('a','b','c'); ou $arr = ['a','b','c'];
$arr[] = 'd'; Ajoute la valeur d à notre tableau
print_r($arr);

$a[] = array('une valeur'). Le $a[] crée un tableau, et la fonction array crée un tableau également. On obtient un tableau à 
l'index 0 dans notre tableau $a.

$c[50] = 'test';
$c[] = 'un autre test';
La valeur'un autre test' va être ajouté à l'index 51 de notre tableau. Prend la dernière valeur rencontrée et ajoute à la suite. 

$users = [
	[
		'name' => 'Chuck',
		'age' => 'eternel',
		'ville' => 'texas',
	],
	[
		'name' => 'Steven',
		'age' => '60',
		'ville' => 'texas',
	],
];

Si on veut récupérer la valeur de l'age de Steven :
echo $users[1]['age']; 
Récupère les valeurs de l'index 1 puis récupère la valeur dont le nom de la valeur est 'age'.

- Conversion de typage (caster)
$str = (int)'125';
$str = (string)$str;
Le (int) convertit le string en int. 
Reconvertit la variable $str en string

$str ='123partez';
echo 5 + $str;
Cela va renvoyer la valeur 128, va parcourir les nombres, supprimer les caractères et ajouter 5.

- SESSION 

session_start();
La session permet de se souvenir de l'utilisateur et de ses informations. 
La fonction va créer et ouvrir une session. 
En créant un fichier coté serveur, et un cookie coté client, le serveur peut identifier vos informations, vos achats lors de la connexion
/!\ à mettre le session_start en premier; (sans saut de ligne, espace ou autre)
La session sera fermée à la fermeture du navigateur.

Ne jamais enregistrer de mdp dans les cookies

- Affichage des erreurs
aller dans le dossier /etc/php/7.2/fpm/php.ini

Modifier le display_errors qui n'est pas en commentaire (recherche avec le "/" + entrée et n pour next)
Redémarrer le php avec service php7.2-fpm restart
Les erreurs seront à présent affichées.
